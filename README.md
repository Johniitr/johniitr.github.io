This repository contains the source code for my academic research portfolio and blog, hosted at **[johniitr.github.io](https://johniitr.github.io)**.

The site is built using **Hugo** (Static Site Generator) and the **Hugo Blox (Academic)** theme, featuring Tailwind CSS for styling.

## Tech Stack

* **Engine:** Hugo (Extended)
* **Styling:** Tailwind CSS
* **Deployment:** GitHub Actions -> GitHub Pages
* **Models featured:** GEOtop, GEOframe
